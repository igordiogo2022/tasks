# Tasks📋

### 📄Descrição  
Gerenciador de tarefas com agrupamento em categorias, organização simples e prática.

<hr>

### 🎮Funcionalidades  
- Criação de tarefas  
- Marcar tarefas como concluídas  
- Remoção de tarefas  
- Interface simples e intuitiva  
- Atualização dinâmica da lista  
- Agrupamento em categorias 

<hr>

### ❓Como funciona  
O usuária cria uma categoria e nela pode adicionar tarefas. As tarefas são exibidas em forma de lista e podem ser marcadas como concluídas, conforme a necessidade do usuário, ele pode deletar as tarefas já concluídas com apenas um click.
